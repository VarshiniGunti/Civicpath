# CivicPath AI - An Intelligent Guide for Indian Voters

A guided assistant that simplifies the election process for citizens across India.

## Problem & Solution

The Indian election process can be overwhelming. Rules for registration, eligibility, and polling are often buried in dense PDFs or scattered across multiple portals. Many voters, especially first-timers, feel unsure about how to start.

I built **CivicPath AI** to replace confusing chatbots with a structured, step-by-step "Voter Journey." By inputting your location and voter type, the system uses Google Gemini to generate a personalized roadmap. Instead of a long conversation, you get a clear, chapter-based guide that tells you exactly how to register, what IDs to bring, and how to cast your vote.

## Tech Stack

- **AI Engine:** Google Gemini 1.5 Flash
- **Frontend:** React, Vite, and Tailwind CSS (Neumorphic Design)
- **Backend:** Node.js and Express
- **Cloud:** Google Cloud Run and Docker

## Key Features

- **Personalized Roadmap:** A custom journey generated by AI based on your specific voter category and state.
- **Interactive EVM Simulator:** A hands-on visual tool to practice using an Electronic Voting Machine and VVPAT.
- **Civic Quiz:** A quick knowledge check with tri-color progress tracking to test your election IQ.
- **Voice Assistance:** Built-in support for speech-to-text and reading out guide chapters for better accessibility.
- **Official Resources:** Verified, direct links to ECI and state-specific voter portals.

## Setup Instructions

1. **Clone the project:**

   ```bash
   git clone https://github.com/VarshiniGunti/Civicpath.git
   cd Civicpath
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Configure Environment:** Create a `.env` file in the root and add your key:

   ```text
   GEMINI_API_KEY=your_google_ai_studio_key
   ```

4. **Run Locally:** Use `npm run dev` to start the app.
